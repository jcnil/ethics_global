Descripción de la Prueba Técnica:

Objetivo: Desarrollar una aplicación o script capaz de encriptar y desencriptar mensajes utilizando un algoritmo de cifrado elegido por el candidato (por ejemplo, AES, RSA, etc.), y que además sea capaz de medir y reportar su propio rendimiento en términos de eficiencia y tiempo de ejecución.

Requisitos:
Selección del Algoritmo de Cifrado: El aspirante debe seleccionar un algoritmo de cifrado simétrico o asimétrico, justificar su elección y describir brevemente cómo funciona.

Implementación de Encriptación y Desencriptación:

El script debe permitir al usuario introducir un mensaje de texto para encriptar.
Tras la encriptación, debe mostrar al usuario el mensaje cifrado.
Permitir al usuario introducir el mensaje cifrado y desencriptarlo, mostrando el mensaje original.
Aislamiento de Indicadores para Benchmarking:

Implementar funcionalidades para medir el tiempo de ejecución del proceso de encriptación y desencriptación.
Medir el uso de recursos del sistema (por ejemplo, CPU, memoria) durante la ejecución del script.
Generar un reporte sencillo que muestre los indicadores de rendimiento del script.
Documentación:

Incluir un README con instrucciones para ejecutar el script, incluyendo los requisitos de sistema y cómo instalar cualquier dependencia.
Documentar el código fuente para explicar las decisiones de diseño y los algoritmos utilizados.

Entregables:

Código fuente del script o aplicación.
Documentación técnica y de usuario (README).
Reporte de benchmarking.

I had the same problem, so I tried changing the permissions and some other commands I found online that messed up my MongoDB installation. Following are the commands that I tried from online and screwed up my installation:

$ sudo chown mongodb:mongodb /tmp/mongodb-27017.sock

$ sudo systemctl status mongod

$ sudo systemctl start mongodb

$ sudo systemctl unmask mongodb

After that, I reinstall mongodb by using following commands, which solved the issue:

$ sudo service mongod stop

$ sudo apt-get purge mongodb-org*

$ sudo rm -r /var/log/mongodb

$ sudo rm -r /var/lib/mongodb

$ sudo apt-get install -y mongodb-org

Now start the mongodb and check the status using following commands:

$ sudo systemctl start mongod

$ sudo systemctl status mongod
